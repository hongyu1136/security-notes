# API 版本降级

## 原理

v2 接口不返回密码，v1 忘了删——切版本号拿到不该有的字段。

```
GET /vulnerabilities/api/public/index.php/v2/user/  → 无 password
GET /vulnerabilities/api/public/index.php/v1/user/  → 含 password ✅
```

![](99_Attachments/图片/1.Low/file-20260724140916877.png)
## 利用

浏览器直接访问 v1 地址，JSON 里多出 `password` 列。

## 费曼版

你公司换了新门禁（v2），但隔壁老侧门没封（v1）——推开就进，老门旁边还贴着员工密码表。

## 危害

密码哈希泄露 → 离线爆破 → 登录任意账号。

## 防护

旧版 API 下线。保留所有版本的话用字段白名单控制输出：

```php
$safe_fields = ['id', 'name', 'level'];
return array_intersect_key($data, array_flip($safe_fields));
```
