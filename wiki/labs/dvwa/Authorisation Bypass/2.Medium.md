## 源码

Medium 在页面加了检查：
```php
if (dvwaCurrentUser() != "admin") {
    print "Unauthorised";
    http_response_code(403);
    exit;
}
```

但两个 API **还是不拦**——`get_user_data.php` 只在 high 检查，`change_user_details.php` 只在 impossible 检查。

## 绕过

页面被挡住了，但可以直接 POST 到 `change_user_details.php`：

```json
POST /vulnerabilities/authbypass/change_user_details.php
{"id":"1","first_name":"hacked","surname":"by_gordonb"}
```

或者直接 GET `get_user_data.php` 拿用户列表。

## 教训

页面权限 ≠ API 权限。每个端点必须独立验证。
