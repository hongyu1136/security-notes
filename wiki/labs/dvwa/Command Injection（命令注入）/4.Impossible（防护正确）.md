
## 查看源码
```php
$octet = explode(".", $target);                    // 用 . 拆成四段
if (is_numeric($octet[0]) && ... && sizeof($octet)==4) {
    $target = $octet[0].'.'.$octet[1].'.'.$octet[2].'.'.$octet[3];
    shell_exec('ping  ' . $target);
} else {
    echo 'ERROR: invalid IP';
}
```
**白名单思路。 不是「过滤危险字符」，是「检查输入是不是纯数字点分十进制」。不是 IP 格式直接拒绝，是就只跑 ping。这才是正确做法。**
