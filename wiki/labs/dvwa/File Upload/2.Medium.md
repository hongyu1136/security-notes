# Medium

## 源码

```php
$uploaded_type = $_FILES['uploaded']['type'];   // 从 HTTP 头取 MIME

if( ($uploaded_type == "image/jpeg" || $uploaded_type == "image/png") &&
    ($uploaded_size < 100000) ) {
    move_uploaded_file(...);
}
```

只检查 MIME 类型，不查后缀。

## 攻击

Burp 抓上传包 → 文件名保留 `shell.php` → **改 `Content-Type` 为 `image/jpeg`**。绕过。

## 为什么能绕过

| 防护 | 为什么没用 |
|------|-----------|
| MIME 检查 | `Content-Type` 是客户端可控的 HTTP 头，Burp 随便改 |
| 文件大小 | 一句话木马 < 100KB |
| 没检查扩展名 | `.php` 直接放行 |
