# Impossible

## 源码

```php
$configFileNames = ['include.php', 'file1.php', 'file2.php', 'file3.php'];

if( !in_array($file, $configFileNames) ) {
    echo "ERROR: File not found!";
    exit;
}
```

## 为什么无法绕过

**硬编码白名单。** 只有四个文件能通过 `in_array` 精确匹配。`file://`、`file1/../`、`..././` 全部不等于这四条 → 直接拒绝。

## 核心教训

防文件包含的正确姿势：
1. 白名单，不是黑名单
2. 精确匹配，不是通配符
3. `in_array` + 硬编码文件名 = 无解
