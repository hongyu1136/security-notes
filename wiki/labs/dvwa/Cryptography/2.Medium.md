## 源码

```php
$key = "ik ben een aardbei";
$decrypted = openssl_decrypt(hex2bin($token), 'aes-128-ecb', $key, OPENSSL_PKCS1_PADDING);
$user = json_decode($decrypted);
if ($user->user == "sweep" && $user->ex > time() && $user->level == "admin")
```

验证条件：用户名=sweep、未过期、level=admin。

## ECB 漏洞

ECB 每 16 字节独立加密 → 相同明文=相同密文 → 不同 token 的块可互换：

| 需要的属性 | 从哪拿 | hex 块 |
|------|------|------|
| 用户名 sweep | Sweep 第0块 | `3061...074c1` |
| 有效时间戳 | Soo 第1块 | `174d...becef` |
| admin 权限 | Sooty 第2块 | `837d...57630` |

拼在一起 → 伪造的 admin token。

## 教训

ECB 模式不安全——块之间无关联。生产环境必须用 CBC 或 GCM。
