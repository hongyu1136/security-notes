## 源码

```php
if (strpos($_GET['redirect'], "info.php") !== false) {
    header("location: " . $_GET['redirect']);
}
```

必须包含 `info.php` 才放行。

## 绕过

把 `info.php` 塞进参数里——`strpos` 只看有没有，不关心是路径还是参数：

```
/source/high.php?redirect=https://evil.com/?a=info.php
```

## 三关总结

| | 过滤 | 绕过 |
|------|------|------|
| Low | 无 | 任意 URL |
| Medium | 拦 `http://` `https://` | `//` 协议相对 URL |
| High | 必须含 `info.php` | 把 `info.php` 当参数值塞进去 |

## 教训

白名单验证跳转目标——只允许已知 ID，不接受 URL。
