# Impossible

## 源码 (`impossible.php`)

```php
// 改为 POST，不再是 GET
if( isset( $_POST['Login'] ) && isset($_POST['username']) && isset($_POST['password']) ) {

    checkToken($_REQUEST['user_token'], $_SESSION['session_token'], 'index.php');

    $user = mysqli_real_escape_string($db, stripslashes($_POST['username']));
    $pass = md5(mysqli_real_escape_string($db, stripslashes($_POST['password'])));

    $total_failed_login = 3;    // 允许失败 3 次
    $lockout_time       = 15;   // 锁定 15 分钟
    $account_locked     = false;

    // 1. 先查失败次数
    $data = $db->prepare('SELECT failed_login, last_login FROM users WHERE user = (:user) LIMIT 1;');
    $data->bindParam(':user', $user, PDO::PARAM_STR);
    $data->execute();

    // 2. 失败 >= 3 次 → 检查是否还在 15 分钟锁定期
    if ($data->rowCount() == 1 && $row['failed_login'] >= $total_failed_login) {
        if (time() < strtotime($row['last_login']) + ($lockout_time * 60)) {
            $account_locked = true;
        }
    }

    // 3. PDO 预处理查库
    $data = $db->prepare('SELECT * FROM users WHERE user = (:user) AND password = (:password) LIMIT 1;');
    $data->bindParam(':user', $user, PDO::PARAM_STR);
    $data->bindParam(':password', $pass, PDO::PARAM_STR);
    $data->execute();

    if ($data->rowCount() == 1 && $account_locked == false) {
        // 登录成功 → 重置 failed_login = 0
        // ...
    } else {
        // 登录失败 → failed_login + 1
        sleep(rand(2, 4));  // 失败随机延时 2-4 秒
        $db->prepare('UPDATE users SET failed_login = (failed_login + 1) WHERE user = (:user) LIMIT 1;');
        // ...
    }

    // 更新 last_login 时间
    $db->prepare('UPDATE users SET last_login = now() WHERE user = (:user) LIMIT 1;');
}
generateSessionToken();
```

## 比 High 多了什么

- **方法改为 POST**（GET 参数直接废了）
- **PDO 预处理** — SQL 注入彻底堵死
- **失败计数 `failed_login`** — 累计 3 次触发锁定
- **锁定 15 分钟** — 期间正确密码也登不上
- **`sleep(rand(2, 4))`** — 随机延时防时间差判断
- **`last_login` 时间戳** — 记录最后尝试时间

## 为什么无法绕过

| 防线 | 怎么堵的 |
|------|---------|
| PDO 预处理 | SQL 注入无能为力 |
| 失败计数 + 锁定 | 3 次后等 15 分钟，字典爆破不可能 |
| POST 请求 | GET 参数直接失效 |
| user_token | CSRF 防护（High 级已加入） |
| 随机延时 | 无法通过响应时间判断 |
