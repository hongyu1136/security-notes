# Low

## 源码 (`low.php`)

```php
$user = $_GET['username'];
$pass = $_GET['password'];
$pass = md5($pass);

$query = "SELECT * FROM `users` WHERE user = '$user' AND password = '$pass';";
$result = mysqli_query($GLOBALS["___mysqli_ston"], $query);
```

## 做了什么

- GET 方式传参，无 token，无验证码，无次数限制
- 密码用 MD5 哈希后才查库

## 攻击步骤

1. 随便输入一个密码，Burp 抓包
2. 右键 → Send to Intruder
3. Positions：清除所有变量，只给 `password` 参数加 `§`
4. Payloads：加载密码字典
5. Start Attack → 看 Length 列，与众不同的就是正确密码

## 为什么能绕过

无任何防护。
**SQL 注入也能打——`$user` 和 `$pass` 直接拼接，万能密码 `' or 1=1#` 同样生效。**

## 登录成功标志

- 响应中出现 `Welcome to the password protected area`
- 响应 Length 与失败时不同
