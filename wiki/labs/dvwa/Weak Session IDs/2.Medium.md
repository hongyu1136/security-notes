## 源码

```php
$cookie_value = time();
setcookie("dvwaSession", $cookie_value);
```

Session ID = Unix 时间戳（秒级）。

## 问题

时间戳已知——所有人看到的 `dvwaSession` 都是当前时间 ± 几秒。攻击者拿到时间戳后前后枚举几十秒就能踩中别人。

## 利用

```
你的 Cookie: 1784784574 (13:29:34)
试 1784784573 → 别人的 session
试 1784784572 → 另一个人的
...
```

## 教训

`time()` 不是随机数，精确到秒的时间范围太容易枚举。
