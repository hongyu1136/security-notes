# High

## 源码

```php
$id = $_COOKIE['id'];                    // Cookie 传参，不是 GET/POST
$query = "... WHERE user_id = '$id';";    // 字符串型
try { $result = mysqli_query(...); }      // 吞掉报错
catch (...) { $result = false; }

if ($exists) → "User ID exists"
else → 随机概率 sleep(2-4) + "User ID is MISSING"   // 时间盲注不可靠
```

## 和 Low/Medium 的区别

| | Low | Medium | High |
|------|-----|--------|------|
| 传参 | GET | POST | **Cookie** |
| 引号 | 字符串型 | 无引号（数字型） | 字符串型 |
| 可用技术 | 布尔/时间 | 布尔/时间 | **只能布尔**（随机 sleep 干扰） |
