# Medium

## 源码

```php
$id = $_POST['id'];
$id = mysqli_real_escape_string($id);
$query = "SELECT ... FROM users WHERE user_id = $id;";  // 数字型，没引号
```

## 和 Low 的区别

- POST 接收，`real_escape_string` 转义
- 但查询是数字型——转义引号没用，payload 不需要引号

## payload

```
1 or 1=1#   → EXISTS
1 and 1=2#  → MISSING
```
