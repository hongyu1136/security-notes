## 逻辑

```javascript
function generate_token() {
    var phrase = document.getElementById("phrase").value;
    document.getElementById("token").value = md5(rot13(phrase));
}
generate_token();  // 页面加载时跑，phrase 默认 "ChangeMe"
```

服务器验证：`token == md5(str_rot13("success"))`。


## 问题

phrase 改成 "success" 后 token 没更新——还是 "ChangeMe" 加密值。

## 绕过

F12 Console 直接算：

```javascript
md5(rot13("success"))
// → "38581812b435834ebf84ebcc2c6424d6"
```

phrase=success，token=上面这个值，提交。

可以F12
```
document.getElementById("phrase").value = "success";
generate_token();
document.forms[0].submit();
```
